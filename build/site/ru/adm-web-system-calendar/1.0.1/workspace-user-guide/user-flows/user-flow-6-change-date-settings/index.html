Изменение параметров дат в календаре

Чтобы изменить параметры дат в календаре:

  1. Перейдите на главную страницу раздела Календари.

  2. Нажмите на значок в столбце Действия и строке для выбранного календаря. Откроется выпадающее меню.

  3. Выберите Открыть. Откроется страница календаря.

  4. Нажмите на выбранную дату. Откроется окно Настройка дня.

  5. Установите флажок Выбрать диапазон, если необходимо изменить параметры для диапазона дат, и укажите дату окончания диапазона.

  6. Установите флажок Выходной для назначения выходного дня для выбранной даты или диапазона дат.

  7. Введите текстовое описание для выбранной даты или диапазона дат.

  8. Нажмите Сохранить для сохранения изменений или Отменить для отмены сохранения изменений.
